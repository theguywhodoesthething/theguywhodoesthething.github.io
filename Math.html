How to use the Math class

The 'Math' class in Java is a static class that contains many useful methods for doing mathmatical calculations beyond what can be accomplished with the simple arithmatic operators. For a class to be static just means that no local changes can be made to the funcionality of that class. There is no way to edit 'Math' so that 'Math.abs()' returns a negative number, for instance. A popular tool in the 'Math' library is 'Math.random'. Math.random() will return a random number between 0 and 1, and is useful when you want to add randomness to your program that isn't controlled by either you or your user. 'Math' contains other useful methods as well, including 'pi' and 'e', methods for finding sine, cosine, and tangent, and logarithmic functions.
